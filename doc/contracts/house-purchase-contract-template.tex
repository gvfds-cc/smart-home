% ====== 第三条: 价款与支付 ======
\section{房屋价款与支付方式}

甲乙双方协商一致，该房屋成交总价为人民币 {{totalPriceCN}} 元整（\textyen\ {{totalPrice}} 元）。

乙方采取以下第 {{paymentMethod}} 种方式支付：

\begin{enumerate}[label=(\arabic*),nosep]
  \item 一次性付款：乙方于 {{payDate}} 前支付全部房价款。
  \item 分期付款：首付款 \textyen\ {{downPayment}} 元于 {{downPayDate}} 前支付，尾款 \textyen\ {{balance}} 元于 {{balancePayDate}} 前支付。
  \item 按揭贷款：首付款 \textyen\ {{downPayment}} 元，贷款 \textyen\ {{loanAmount}} 元。
\end{enumerate}

% ====== 第四条: 税费承担 ======
\section{税费承担}

本次交易产生的契税、增值税、个人所得税、印花税、不动产登记费等，由甲乙双方按规定各自承担。

% ====== 第五条: 交付与过户 ======
\section{房屋交付与产权过户}

甲方应于 {{deliveryDate}} 前将房屋交付给乙方，结清交付日前的水、电、燃气、物业等费用，并配合办理户口迁出手续。

甲乙双方应于 {{transferDate}} 前共同向不动产登记机构申请办理产权转移登记。

% ====== 第六条: 违约责任 ======
\section{违约责任}

一方逾期履行义务的，每逾期一日按房屋总价款的万分之三向对方支付违约金。因一方原因导致合同无法履行的，守约方有权解除合同并要求赔偿。

因不可抗力导致合同无法履行的，双方互不承担违约责任。

% ====== 第七条: 争议解决 ======
\section{争议解决}

本合同履行中发生争议，双方协商解决；协商不成的，向房屋所在地人民法院提起诉讼。

% ====== 第八条: 其他 ======
\section{其他约定}

本合同自双方签字之日起生效，一式两份，甲乙双方各持一份，具有同等法律效力。
